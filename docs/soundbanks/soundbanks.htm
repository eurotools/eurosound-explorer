 Soundbanks

v1.00 Updated: 07/01/23

Each level in EngineX is linked to a soundbank, these type of files contain the major sounds that are needed by this level: monsters, hud, objects etc.. There are three panels that compose the soundbanks exploration: Soundbank Hashcodes, Sample Pool, Sample Properties, Wav Header data.

 


Panels

  1. Soundbank HashCodes
  2. SFX Properties
  3. Sample Pool
  4. Wav Header Data
